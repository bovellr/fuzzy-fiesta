## Power BI semantic model review download

The Power BI semantic model review is available in two locations:

- [`POWERBI_SEMANTIC_MODEL_REVIEW.md`](POWERBI_SEMANTIC_MODEL_REVIEW.md) — root-level copy for easy download from GitHub.
- [`docs/powerbi_semantic_model_review.md`](docs/powerbi_semantic_model_review.md) — documentation-folder copy.

To download the Markdown file from GitHub, open `POWERBI_SEMANTIC_MODEL_REVIEW.md`, click **Raw**, then save the page as a `.md` file.
